G'day all,
I released a great new program to communicate between serial ports.
It can send files, ASCII Artist pictures (with viewing) and has a nice
chat function.

Files save to the disk with the same name (if it exists you will be
asked for a new one).
They are limited to 6000 bytes, so you may need to split up big files.

There's also a send picture function that allows you to have the an ASCII Artist
picture appear on the screen of the receiver, which you can save or discard.

Lastly there a chat function, it uses a custom input function so you
can receive
messages while you are halfway through typing one. Use '#exit' to exit
from chat.

Try it out and let me know what you think!
http://code.google.com/p/mikebasic-applications/downloads/detail?name=SERIAL.BAS&can=2&q=

P.S. I update ASCII Artist to 2.2.2 to fix some file saving problems.
http://code.google.com/p/mikebasic-applications/downloads/detail?name=draw.bas&can=2&q=
