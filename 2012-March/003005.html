Ignore my code because that incorrectly uses
the memory access, it should be
mov word [gs:0x0070], ax
instead of
mov [0x0000:0x0070], ax

That's the Interrupt Vector Table (IVT) in real mode.
The Interrupt Description Table (IDT) is for
protected mode. It's easy to remap interrupts.
To find the location just multiply the interrupt
number by four. This will locate a four byte
segmented address formatted, Segment:Offset.

Here's a link to the OSDEV artical:
http://wiki.osdev.org/IVT
