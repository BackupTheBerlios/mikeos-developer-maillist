Please, let's not talk about this. Porting Python to a generic,
32-bit, POSIX-like OS is very tough work. Porting it to a 16-bit

impossible, especially for a 15 year old who is still learning asm (no
offence!).

interpreter for a tiny subset of Python, but talking about a generic
Python port is like talking about Firefox or OpenOffice.org. Not going
to happen, I'm afraid.
