I release a minor update to the level editor that allows you to see
the data values of the currently selected item in hexadecimal.
It is not ideal but you can see what values all the fields have.
The actual game is, so far only a couple hundred lines of assembly.
http://code.google.com/p/mikebasic-applications/downloads/detail?name=lvledit.bas&can=2&q=

Pro Tip - If you select "New Level" from the Save/Load section of the
main menu, you can choose what character you want the background
to be filled with.
