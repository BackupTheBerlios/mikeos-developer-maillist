It's taken long enough, but we finally have a shell that works! For those of
you who want to see the source code, head to
http://apolloopsys.sourceforge.net/ and click on 'Latest Source Code'.
Alternatively we have a zipped version here:
http://apolloopsys.googlepages.com/apollodev.zip

There's some interesting stuff there, including some bug fixes, less of a
countdown, improved documentation and a shell that recognises commands. It's
not quite a version 0.0.7 release yet as I still need to code the commands
themselves, but for those of you who are interested it should give an idea
of what's to come. :-)
