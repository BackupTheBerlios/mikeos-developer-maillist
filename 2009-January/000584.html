Hi.

I have released TomOS-0.1

http://surfnet.dl.sourceforge.net/sourceforge/tomos/tomos-0.1.tar.gz

Main changes are compare to MikeOS-2.0.0:

# changed directory organisation.

# every system call is called by int 47h. I have moved all system calls
  to work under int 47h.

# new documantation and simple handbook. see doc/index.html and
  programs/examples.html

# written perl program to generate documentation from source
  code. see tools/docgen and doc/DOCGEN.txt

# removed all MikeOS programs because TomOS is not compatible with
  MikeOS at the moment. I working on solving this problem.

# I wrote few example programs to demonstrate how to write programs
  for TomOS. see the programs directory. 

If you find a while to look at it and point out mistakes, errors and
suggestions I would appresiate it very much.
