Hello,

Here is tested library, libmikeos-0.2.0 for MikeOS-3.x series :-

http://www.bi.a.u-tokyo.ac.jp/~uaa/gomitext/2009/20090112/libmikeos-0.2.0.tar.gz

The differencies from untested version (20081213.tar.gz) are :-

	- removed unused API glue (mouse/interrupt handler)
	- added MikeOS-3.1 API glue (os_string_charchange, os_draw_block,
	  and os_get_random)
	- fixed the bug of os_filename_convert/os_get_time_string glue
	- samples are modified
		test03 (string operations)
			os_string_charchange test added
		test05 (misc #1)
			os_draw_block and os_get_random test added
		test07 (interrupt handler)
			removed
		test09 (file operation #1)
			the target of os_file_selector is changed
			MIKEKERN.BIN -> HELLO.BIN, and the result of API
			requires "MIKEKERNBIN" -> "HELLO.BIN"
		test0c (mouse #1)
			removed
		test0d (mouse #2)
			removed
