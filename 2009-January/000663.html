Lo all,

I've been doing some work on what will become 3.2.1. The changelog so far:

# CLI now includes 'CAT' command to display a file: 'CAT FILE.TXT'.
This can be used to quickly see the contents of a text file without
having to run VIEWER.BIN

# CLI code (source/features/cli.asm) cleaned up -- split into sections
with fewer local labels

# os_string_parse call is now accessible to external programs

# API version bumped to 11

# Fixed comment typos

So it's a bunch of small changes; I'm going to keep going through the
code and making sure that everything is as clear and well explained as
possible. If anyone has ever spotted any typos or confusing comments
in the code before, let me know and we can make 3.2.1 the most solid
and clearly documented release yet!
