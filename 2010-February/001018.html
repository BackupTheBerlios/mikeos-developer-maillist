Hi Mike,

What do you think about adding the following commands/functions to your BASIC
which would then allow more flexibility especially when calling BIOS int's:

1) INTS.ON   enables h/ware interrupts -  sti
2_ INTS.OFF disables h/ware interrupts - cli

Then for registers, something like this:

To get the value in a register:

a = (reg.AX)  or a = (reg.AH)  or a = (reg.AL) etc.

To set a value in a register:

reg.AX = a  etc.

And finally, INT(xxh) to call a BIOS interrupt.

In this way one could from BASIC populate some regs and do a BIOS call:
An example to read character and attribute at cursor position:

reg.AH = 8
reg.BH = 0
INT(10h)
a = (reg.AL)   ; character returned in al
b = (reg.AH)   ; character's colour returned in ah
