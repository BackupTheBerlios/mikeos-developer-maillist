Lo,

Paulo asked what he can do to help out with BasicOS, so I'll paste my
reply here as it might be useful for the list:

Anyone can help right now -- there's loads of stuff the BASIC interpreter needs:

- Support for arrays (DIM, using array1(8) for eighth element etc.)
- String comparisons (eg does string 1 contain string 2?)
- Printing text with colour
- User-defined variable names (not just A - Z)

that's just off the top of my head. These features can be added to
MikeOS BASIC and then they'll become part of BasicOS one day.

I appreciate the effort with logos and stuff, but really, by far and
away we need code, code and more code!
