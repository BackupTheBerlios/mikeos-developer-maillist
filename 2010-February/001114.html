Hi Cliff,

If you're a regular coder in other languages, you might get some use
out of this (apologies if you've already seen it):

http://mikeos.berlios.de/write-your-own-os.html#asmprimer

It's short but it'll get you familiar with the concepts. Otherwise
tutorials are (and have to be) very specific to the platform you're
running. If it's Linux/UNIX then there's some good stuff here:

http://asm.sourceforge.net/resources.html#tutorials

If you're on Windows, I recommend grabbing VirtualBox and an Ubuntu
ISO -- it's a doddle to install thesedays and it's really easy to get
OS dev tools. Unix lets you do useful stuff with disk images etc.
out-the-box, whereas Windows needs all sorts of add-on tools.
