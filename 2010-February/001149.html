Hi nascent mind,

Depending on the results you get from running the exe with -writeromfile
you have two ways to proceed:

1) If it gives you an error then download CBROM32_195.exe and with this you can get all the gory details of the bios.
2) Use DEBUG (much easier)

For example start DEBUG then type in D F000:FFE0 and enter.
You should get the manufacturer, version and date of your bios if memory servers me correctly.
By poking around with different addresses you can get all the bios data.
Keep in mind that some bios code is 32 bit and this will trip up DEBUG so rather save the "dump" from DEBUG
in 32KB chunks like Bazza said then run it thru ndisasm.
There are plenty of tuts on the net on how to save files with DEBUG.

Paulo.
