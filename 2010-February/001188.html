If you need more precision or no rounding off, then set the
configuration register of the x87 to enable either single or double precision floating point numbers
and use different opcodes such as FLD/FLID and FSTP/FISTP.
You would also have to allocate more memory for the variables i.e. Tbyte.
Of course the result would then have to be converted back to "normal" numbers.
