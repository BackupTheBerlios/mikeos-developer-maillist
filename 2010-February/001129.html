Hi nascent mind,

That is strange being all in an exe.
I did a quick google around and most of the manufactures supply the exe which is
actually the flashing utility but they also provide the bios contents in a separate file
which normally has a .rom or .bin extension.

I quickly downloaded an arbitrary bin file for some Asus motherboard and ran the bios bin file thru ndisasm (part of nasm)
and I got the code.
The most interesting is:

000FFFF0  EA5BE000F0        jmp word 0xf000:0xe05b

Just like Bazza said a few posts ago.

Why exactly do you want to "look" at the bios code?
Do you have a problem with yours, in which case I suspect you've come to the wrong place
or do you just want to use the code for learning purposes in which case grab any old one from the net.

Paulo.
