I'm assuming you are using a pretty recent machine so when you are using text mode (03h)
your s/vga card is actually emulating an ega screen og 80 X 25 X 16 colours.
When in graphics mode (13h) it's in 320 X 200 X8 VGA mode.
According to the VESA standards, svga is anything of 800 X 600 and up.

So the simple answer is....no.
