> Keep in mind that not all USB flash keys are created equal.

MikeOS does it differently to other OSes: we direct write the floppy
image to the start of the USB key, and then the PC boots it as a
virtual floppy drive. Which leads to...

> How would one be able to save the a file to the flash disk?

Brilliantly, thanks to this BIOS floppy emulation, we don't need to
know anything about USB, FAT16 etc. We just write data using the
regular BIOS floppy drive write-sector interrupts and it's all handled
automatically.

So in other words, NEWBASICOS would, like MikeOS, be a floppy disk OS,
but usable on USB keys thanks to BIOS emulation. The more people who
test on USB keys the better:
http://mikeos.berlios.de/handbook-user.html#writing

'dd' does the job on Linux/Unix, and Flashnul on Windows. It's worked
with all combinations of USB keys and machines I've tried so far
(albeit not many though!)
