The workings of the BASIC interpreter are quite simple:

The core is the get_token routine, which returns a 'piece' of the
BASIC code (ie a number, quote, variable etc.) After you call
get_token, AX contains the type -- eg VARIABLE, QUOTE, STRING VAR etc.

The interpreter looks at the first item in a line, and jumps to the
corresponding routine for the instruction (eg if it's GOTO we jump to
do_goto). Then do_goto, as with the other instruction routines, calls
the tokenizer to get chunks and work out what to do with them.

When we're done with the line, we jump back to 'mainloop' where the
next line is processed.
