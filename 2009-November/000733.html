Hi all,

http://prdownload.berlios.de/mikeos/mikeos-4.0rc2.zip

This fixes a couple of bugs in the file manager and BASIC interpreter,
and (rather inappropriately for a Release Candidate, I admit) adds a
new instruction to BASIC: 'CURSORCHAR'.

This stores into the following variable the character underneath the
text cursor. With this, I've been able to write a simple one-player
Tron-like game -- run EXAMPLE.BAS to try it out. You can also see it
at the end of the updated Handbook:

http://mikeos.berlios.de/handbook-appdev-basic.html#miketron

Now, honestly, cross-my-heart, this is the proper Release Candidate --
unless we find any more bugs, this'll be the final 4.0 version at the
weekend :-)

Cheers,
