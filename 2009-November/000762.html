Hello, libmikeos-4.0 has released.

http://www.bi.a.u-tokyo.ac.jp/~uaa/gomitext/2009/20091122/libmikeos-4.0.tar.gz

The differences between libmikeos-3.3 and libmikeos-4.0 are :-

- The argument of mikeos_pause() has changed.
  3.3) void mikeos_pause(unsigned long microseconds);
  4.0) void mikeos_pause(unsigned int deciseconds);

- New API wrapper to call os_run_basic, currently return value is zero.
  4.0) int mikeos_run_basic(char *buf, int size);

- Fixed the bug of mikeos_int_to_string(), os_int_to string wrapper.

- The version number in miksos_defs.h has changed.
  3.3) LIBMIKEOS_MAJOR_VERSION 0x0003
       LIBMIKEOS_MINOR_VERSION 0x0003
  4.0) LIBMIKEOS_MAJOR_VERSION 0x0004
       LIBMIKEOS_MINOR_VERSION 0x0000

And, some samples are rewrited.

- test01.c, test05.c
  All mikeos_pause(500000UL) are changed to mikeos_pause(5).

- test09.c
  This sample was depended on HELLO.BIN, but now requires KERNEL.BIN.

- test0c.c
  New sample, it runs EXAMPLE.BAS from C code.

Please be careful the change of mikeos_pause(), and sorry for the bug of
the wrapper for os_int_to_string.


Have fun!
