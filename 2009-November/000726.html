Hi to those still lurking on this list!

http://prdownload.berlios.de/mikeos/mikeos-4.0rc.zip

It's been a while, but in the last week I've done a lot of work on the
BASIC interpreter. It has now 23 commands, including string handling,
loading/saving files, moving the cursor around (and turning it off),
PC speaker sound, and nested FOR... NEXT loops. Oh, and no longer do
you have to use numbers as labels -- you can use 'foo:' etc.

Here's the full manual -- see the end for some sample programs
demonstrating various features:

http://mikeos.berlios.de/handbook-appdev-basic.html

In other news, the EDIT.BIN editor has been improved. Backspace now
works (Delete still required to erase newlines), while the keybindings
have been changed:

* Esc = quit without saving
* F2 = save file
* F5 = remove current line (good for quickly removing chunks of a file)
* F8 = execute edited code in the BASIC interpreter

Yes, you can run your code as you're editing it! Welcome to the MikeOS
IDE :-) This is thanks to a new system call, os_run_basic. See
doc/CHANGES.TXT for more info on this and other updates.

I just need to update some of the docs; otherwise this'll be good to
go at the weekend. Please test it out, and if anyone wants to write
some BASIC programs to test the interpreter and include in the Samples
section of the handbook, that'd be brilliant! The more examples the
better.

Any questions, just ask here on the list and I'll reply ASAP.

Cheers!
