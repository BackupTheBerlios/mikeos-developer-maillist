Lo Bazza,

The video mode is set at the start of source/kernel.asm thusly:

        mov ax, 1003h                   ; Set text output with certain
attributes
        mov bx, 0                       ; to be bright, and not blinking
        int 10h

I used to have a graphical demo program that switched to 320x200 256
colour mode like this:

        mov ax, 13h
        int 10h

And plot pixels with CX = X position, DX = Y pos, and al = colour,
then this code:

        mov ah, 0Ch             ; Plot pixel
        int 10h


It switched back to the standard text mode like this:

        mov ax, 0
        int 10h

        mov ax, 3               ; Normal video mode
        mov bx, 0
        int 10h
        mov ax, 1003h           ; No blinking text!
        int 10h


Hope that helps a bit! I don't know much about the graphics modes but
I managed to get those routines working.
