Hi Mike...

Well the home brew parallel port analogue to digital converter board is now
working in MikeOS... Tricky but not insurmountable. It now gives the ability
to add on to it and use MikeOS as a Voltmeter, Ammeter, Powermeter, Data
logger/Transient Recorder and much more.

It was finally tested on this quad core q6600 machine running at 2.4GHz
with a parallel port.

This is a proper hardware electronic project that can be built by anyone
from about 10 years old on his/her kitchen table.

I had to put two time delays in the assembler code to slow it down as
it was originally designed for MUCH slower gear of yesteryear; modern
machines are much to fast for direct access to a parallel port.

It will have a circuit, parts list, photos, SW, source code and instructions
when finished.

FTTB if you want the source, circuit diagram, parts list and SW, gimme an,
(email?), address to send to and you can have what is done ATM...
