Dear All at MikeOS-Dev,

I've just started with MikeOS, & encountered one slight problem relating to installing Qemu in Ubuntu. My machine has Ylmf installed: Ubuntu 9.10 with an XP-like desktop!

The instruction "sudo apt-get install build-essential qemu nasm", from 'How to write a simple operating system' didn't work in my machine.

However, the following terminal instructions did [don't use the " " marks]:

 "sudo aptitude install qemu qemulator" [enter & wait while it processes the command]

then, at the next $ prompt:

 "sudo aptitude install qemu-launcher" [enter &c.]

then at $:

 "sudo aptitude install qemu nasm" [enter &c.]

All done!

The first two lines came from: http://www.ubuntugeek.com/qemu-machine-emulator-and-virtualizer-setup-in-ubuntu.html , which has further instructions on configuration of Qemu. The final line was a calculated guess!

Hope this of help to anyone else who's encountered this problem.
