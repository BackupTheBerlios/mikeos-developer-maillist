> So, can somebody please help me? I haven't yet found out how to access any
> of these options in assembler.

You won't find five-point step-by-step guides for those things, no,
because they're complicated. Especially Ethernet and TCP/IP. Few hobby
OSes support that. Looking at what you have so far, it's about 5% of
what's required to support those features, so there's a long road
ahead.

Not to put you off or anything -- but don't try to run before you can
walk. Create a FAT12 floppy disk driver, as that'll teach you a lot
about handling hardware, parsing filesystem tables and so forth.

You might find useful code in MenuetOS or KolibriOS, but it's very
unlikely it'll be directly portable to your OS.

Good luck!
