Hey all,

First time post, long time lurker here. I love Mike OS, great project you
have here!

It's been a while since i coded 8086-only assembly, but ... if memory
serves, 8086 only supports short conditional jumps (where displacement is
between -128 and 127). Here you have displacements of -142 and 148, so the
assembler probably outputs the near conditional jump opcode. You may need to
combine conditional jumps with small displacements with unconditional jumps
with 16-bit displacements:

cmp foo
jz f1
jmp f2
f1:
....
f2:

instead of

cmp foo
jnz f2
....
f2:

This might work...

Best,

Mario
