Thanks! I'm really glad you've found it useful. I grew up with the ZX
Spectrum, and its BASIC has no doubt influenced mine in many ways...

One idea I had, a while back, was to make a spin-off version of MikeOS
that booted straight into BASIC. Users could write it to a USB key,
boot it up and explore, like on the old 8-bit machines. People could
give it to their kids, so the young 'uns could get a sense of
controlling the machine, poking and peeking bytes, like how we learnt.

I think there are a few things missing from MikeOS BASIC to make that
doable though: READ + DATA, proper array declaration and handling, and
arbitrary variable names. If I ever get round to implementing them, I
might start BasicOS as a side project and see how it goes!
