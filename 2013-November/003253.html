>Yea, I tried Ubuntu at first but I had issues with just about everything.>In windows, I understand that NASM commands only work when my cmd is in the NASM directory, but in Ubuntu I >couldn't get any NASM commands to work.
As somebody that programs in both Windows and Lubuntu (Ubuntu) I can say both appear to be just as easy :)
>So I was hoping there would be someone knowing how to do it in windows :p
Windows is pretty simple, just drop your NASM files into System 32. You should then be able to type 'nasm' over the command line!
Next thing is making your own bin file. If you are anything like me you may use spaces in folders, so to get around this simply type something like 'nasm "directory/of/program.asm"'. It then creates a binary called 'program' and if it's important to you then rename it to 'program.bin' (which I believe MikeOS expects).
All you then have to do is get your floppy image, mount it using which ever mounting tool is your favourite and then copy your binary files over. I personally used PowerISO to mount my floppy drives and ISOs but everybody to there own.
Also... on my Windows XP machine I used to be able to drag and drop my .asm files over nasm.exe and it would compile it automatically although it does appear they took that out with newer versions, but running some backwards compatibility settings may or may not help you do the same.
Hope that helps.
