The option that works the best for me, after trying a few floppy image programs, is to use a virtual machine installation of either WinXP or 2000. Share my dev folder as a network drive on the virtual machine, drag the files into the A drive, which is loaded with a copy of a MikeOS image, and then go into a program such as MagicISO (I think?) and specify the boot image as that same floppy image.

The reason why I do this is because Windows 7 and all of the image manipulators I've tried do not handle MikeOS's FAT12 correctly. The usually end up changing it into some sort of FAT16 hybrid or something, I dunno. It's very strange.
