> How I can make it run COFF and ELF files?

By writing the code necessary to implement COFF and ELF support.
Google is your friend, as is OSDev.org. ELF is a complex format:

http://wiki.osdev.org/ELF

Read EVERY word on that page (and UNDERSTAND it!) before you decide if
you want to support such a powerful, complicated format in a simple,
real mode OS.

> And is there any way to invent my own file format?

Er, yeah, by inventing it! That's as broad as saying: is there any way
to invent a machine? You choose what the bytes do, and where they go
in the files, and that's it -- you have a file format.
