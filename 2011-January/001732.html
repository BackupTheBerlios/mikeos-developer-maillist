> Hi! I've made a bootloader to load sector 1 (the kernel).
> It don't works. Do you know what is wrong in this code.
> I've attached the bootloader and the kernel.

Cylinder numbering in CH begins at 0, so the first cylinder
is 0, next is 1 etc.

The cylinder numbering is the only reason the code doesn't
work but the following is good to know.

A correct BPB begins with 3 bytes of executable code:

  jmp start
  nop

Don't assume that the IPL is a floppy disk, the BIOS provides
the drive identifier in DL, the USB flash drive is the new floppy
disk - see USB Booting Secrets here:
http://board.flatassembler.net/topic.php?t=12389

A segment value of 1000h is "org 10000h"

There is no need to reset the boot device at the very beginning.
