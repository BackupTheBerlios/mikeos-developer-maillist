> It looks like Takayoshi wrote it on OpenBSD, so I'd definitely try it
> on Linux. With VirtualBox + Ubuntu you can have a Linux installation
> on your Windows desktop for free in half an hour. OS dev can be a pain
> on Windows -- it's easier to get the right tools on Linux.
> 
> > anything. When and if I do get this built do I just drop the 3 files in the
> > lib folder into mikeos programs folder or would they go in features folder?
> 
> Neither; it doesn't say that in the ReadMe. libmikeos is separate from
> MikeOS itself. Look in the sample/test00/ directory for instance, and
> you'll see that running 'make' generates a .bin file -- a MikeOS
> executable. You can then copy it onto the MikeOS floppy disk (or into
> the disk image with loopback mounting).
