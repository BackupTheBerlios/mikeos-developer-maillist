As for Windows saying it's not formatted, the bootloader (for MikeOS)
specifies the file system as FAT12, but the structure of the file
system actually on the disk is FAT32, so the Windows FAT12 driver
can't do anything.  Writing a driver for another filesystem is as bad
as writing a driver for a blank disk.
