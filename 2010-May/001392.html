http://prdownload.berlios.de/mikeos/mikeos-4.1.1-rc2.zip

As with RC1, the only change is:

# os_pause rewritten to use int 1Ah and work better with recent
BIOSes; now uses 110ms intervals instead of 10ths of a second

If folks can test it on real hardware, that'd be great. Fire up
EDIT.BIN, open EXAMPLE.BAS and type this at the top:

  print "pausing..."
  pause 30
  print "done!"
  end

Then hit F8 to run the code. There should be a 3-second-ish pause
between the two words.
