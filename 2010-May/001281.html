Some flash drive formatting utilities use an MBR.
Just don't use it.
We are assembly language programmers. Everything can be done with
a text editor, an assembler and dd. Use dd-removeable to write directly to
the physical flash drive (see previous post).
Don't use utilies to build the ultra simple FAT12 file structure.
See fat.asm in pdBIOS32.

Windows uses write caching with the flash drive.
After dd'ing remove and re-insert the flash drive so that windows can
recognize the file system, then copy files.
