Yeah,

The new boot-loader does everything that the old one did, but...

If the boot loader can't be found, (in this case 'AUTORUN.BIN'), then a
small command line appears, allowing you to load a different one.

This can be useful, if:
- You have more than one kernel
- It failed to load the first time, and you wish to try to reload it
- If you actually want to use it as small base OS, as it can actually
load 'programs', and run them.

To use it, you need to make sure that the filenames you type are in
CAPITALS, and that filenames are equal to 11 characters. E.g. KERNEL.BIN
= KERNEL  BIN.

I know some of it is a bit hard to use, but for using no extra code, why
not have it there?
