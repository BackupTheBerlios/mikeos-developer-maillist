Hi all,

The only change here is that the os_pause routine (in
source/features/misc.asm) has been rewritten based on Paulo's code,
using int 1Ah. This should hopefully work better on modern BIOSes. The
routine behaves very slightly differently to the old one in that it
delays for 110ms chunks instead of 100ms, but that shouldn't break
anything.

http://prdownload.berlios.de/mikeos/mikeos-4.1.1-rc.zip

If folks can test it on real hardware, that'd be great. Fire up
EDIT.BIN, open EXAMPLE.BAS and type this at the top:

  print "pausing..."
  pause 30
  print "done!"
  end

There should be a 3-second-ish pause between the two words. I'm going
to leave the RC file up for a week; if there are no problems, I'll
simply remove the '-rc' from the filename and 4.1.1 is done!

Cheers,
