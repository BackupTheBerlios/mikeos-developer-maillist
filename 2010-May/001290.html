>1) I remember reading somewhere that some BIOS' also require the "enable USB legacy emulation" feature to be set
>   before booting from anything USB.

Well, someone mentioned follow thing, didn't it?

1-a) Modern USB-aware BIOS can specify legacy
emulation type such as CD-ROM, HDD and FDD.
Normally this menu is Auto, but BIOS treats
USB memory stick as FDD when you selected.

If there is no type selection, USB stick
will be recognized as HDD.

I think the trouble comes from the
differencies of Cylinder/Sector/Head paramater
between FDD and HDD.
