Mike,

Reading the system time works.(i.e. number of ticks since last reset - resolution is 18.2 ticks per second or 55 milliseconds)
I have tried this by using the TIME command in the MikeOS CLI and it does work.

The motherboard is an Intel D410PT with BIOS reporting as MOPNV10N.86A and the CPU is an Intel Atom running at 1.6GHz and 512MB RAM.

A quick test to see if a specific BIOS supports INT15h, 86h is to type the following in EDIT.BIN then pressing F8 to run it:

PRINT "press a key to test"
PAUSE 4
PRINT "If you see this message then all is OK"
END
