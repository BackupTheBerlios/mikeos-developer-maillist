Hi fellow MikeOS followers...

This is the very basic MIKE-BASIC code for doing an analogue 7 bit depth
display in CGA text mode.
When I have finished the 8 bit version I will post it on here...
IF you decide to try it out using cut-n-paste, REMEMBER to edit in UNIX style
text before saving it to MikeOS system "disk" and trying it...
There are a few subtleties to do before I use it in anger in a project...
I`m working on ideas for colour ATM using BASIC to tart up this bargraph
generator...

Enjoy... :)



REM 7 bit bargraph display for MikeOS ADC...

REM Variable setup.
REM a = solid character block.
REM b = left hand side half character block.
REM c = 8 bit maximum value from ADC divided by 2 to give pseudo 7 bit.
REM d = right hand side half character block.
REM e = start character for 0 only.
REM k = Esc key value.
REM n = anything from 0 to 256.
REM m = anything from 8 to 72, for the horizontal print position.
a = 221
b = 219
c = 255 / 2
d = 222
e = 179
k = 27
n = 0
m = 8

REM This is the basic analogue bargraph generator slowed down to
REM see it basically working.
CLS
scan:
REM It will still stop without KB intervention.
IF n > 127 THEN GOTO getout

REM The following line added for a ZERO start.
MOVE m 10
IF n = 0 THEN PRINT CHR e ;
IF n = 0 THEN n = n + 1

REM Start with character decimal d above.
MOVE m 10
IF n = 1 THEN PRINT CHR d ;
IF n = 1 THEN m = m + 1
IF n = 1 THEN n = n + 1

REM Carry on with alternate characters decimal a and b above.  
IF n <= c THEN PRINT CHR a ;
n = n + 1

REM Purely a delay to see it working.
PAUSE 5

REM Carry on with alternate characters decimal a and b above.
MOVE m 10
IF n <= c THEN PRINT CHR b ;
n = n + 1

REM A delay just as above.
PAUSE 5

REM Get out clause at any time.
GETKEY k
IF k = 27 THEN GOTO getout
m = m + 1

REM These print statements are not required!
MOVE 22 3
PRINT "Character shift position is:- " ;
PRINT m ;
PRINT "..."
GOTO scan

REM Clean exit.
getout:
PAUSE 30
CLS
END
