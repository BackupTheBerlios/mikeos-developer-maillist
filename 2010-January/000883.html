Hi, there.

libmikeos-4.1 is now ready :-

http://www.bi.a.u-tokyo.ac.jp/~uaa/gomitext/2010/20100119/libmikeos-4.1.tar.gz

There is two new, and two modified functions.

[New]
  unsigned char mikeos_port_byte_in(unsigned short port)
  void mikeos_port_byte_out(unsigned short port, unsigned char data)

[Modified]
  unsigned int mikeos_get_random(unsigned int low, unsigned int high)
  void mikeos_serial_port_enable(int mode)

  former:
  int mikeos_get_random(void)
  void mikeos_serial_port_enable(void)

Please be careful when you port your program on libmikeos-4.0
to libmikeos-4.1.
