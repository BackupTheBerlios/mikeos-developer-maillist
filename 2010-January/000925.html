Yes, modern PC(after Windows XP era)'s BIOS
has "legacy emulation". It enables :

1. act USB keyboards/mice as PS/2 keyboard,
like they are connetcted via PS/2 keyboard
controller.
2. user can access USB storage/floppy discs 
via BIOS (no hardware emulation).

So, I think BIOS wraps difficulties of
handling USB device problem.

And we have to afraid that :

a. Old PC (Windows 98/2000 era) has poor/
unreliable legacy emulation. But they also
have "real" PS/2 and floppy hardware.
No problem.
b. Make a way to boot another device, when
floppy has died.
