Here is some info that might help:

================================================
IBM PC/XT keyboard (1981)
================================================

83 keys
5 pin din connector
Simple uni-directional serial protocol
Uses scan code 1
No host to kb commands

=================================================
IBM AT keyboard (1984) Not backward compatible with XT systems
=================================================

84 - 101 keys
5 pin din connector
Bi-directional serial protocol
Uses scan code 2
8 host to kb commands

=================================================
IBM PS/2 keyboard (1987) compatible with AT but not XT
=================================================

84 - 101 keys
6 pin mini-din connector
Bi-directional serial protocol
Uses scan code 3 (but only scan code 2 is guaranteed for all manufactures)
17 host to kb commands (Acknowledges all commands but may not act on all).

Like I said in my previous post, you can control the PS/2 kb (including the LED's)
via ports 60h and 64h.
