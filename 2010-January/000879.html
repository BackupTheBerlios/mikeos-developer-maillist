Hi Mike at al...

I started a kids level Parallel Port Dev Board some time ago for the AMIGA...

<http://aminet.net/package/docs/hard/PAR_Dev_Board>

This board HAS had much more built on it since this last upload, it now
has a PWM DAC, on board Logic Probe, and a(n) R2R DAC finished...

(The PWM DAC will only work on AMIGA.)

It uses only transistors, passives and diodes - NO ICs!

It only needs an ADC to be built to complete it, AND, yes, NO ICs for
the ADC too... ;-)

As it is UNIVERSAL I have decided to include it as a project for MikeOS.

In doing so I have thought of a way to generate waveforms using this
board and creating an "Arbitrary" AF Function Generator signal source
using some assembler and MIKE-BASIC...

So waddaya reckon guys?
Waveform Generation using MikeOS all inside 32K, boy what power eh! :D

Is it worth going ahead with?

It will take some time and I will be able to include a final update to AMINET
too, "killing two birds with one stone"... :-)
