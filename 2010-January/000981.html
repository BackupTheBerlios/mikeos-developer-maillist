OK, I think I'm starting to understand M$ thinking here.
There seems to be 3 versions (at least) of cmd.exe

The first type is used in XP home.
Do a file properties check on it and look under version, it should be something like 5.1.xxxx.xxxx

Then there is the version used with XP PRO and no sevice packs:
5.1.xxxx.0 xpclient xxxxxx-xxxx

Then the one in XP PRO SP2 and XP MEDIA CENTER EDITION SP2:
5.1.xxxxxx.xxxx xpsp_sp2.rtm.xxxxxx-xxxx

(in all cases the x's represent numbers which seem to vary from pc to pc)

The only version that does not seem to support nul is the one used in XP home.

What cmd.exe version is on your XP box?

If I'm correct then you have two options:

1) go with your idea of redirecting to a text file
2) Since M$ gives SP2 away for free,  I don't see a legal problem with "getting" only the cmd.exe from SP2.
  Of course I'm assuming that SP2 is the same for all versions of XP, but then again it is M$ were are talking about.
