Probably an idea people will hate, but Windows Script Host has proved more reliable for me in the past:
http://msdn.microsoft.com/en-us/library/ms950396.aspx

It's got a bad name because it uses VBScript (or JScript) and is usually associated with viruses due to security issues. But it's considerably easier to understand for new programmers and works on every version of Windows since 98.

Why hack Windows like UNIX when you can speak the lingo? :-)
