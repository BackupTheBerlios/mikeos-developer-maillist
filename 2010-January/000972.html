I think I've figured it out now. "nul" doesn't exist on Windows XP. If
you change the tests to:

  nasm > wut.txt 2> wut2.txt
  if eERRORLEVEL 9009 SET MIKEOS_NONASM=1

Then it works on XP. Of course, you have to remove those .txt files at
the end. (You can't redirect both stdout and stderr to the same file
-- Windows complains.)

Fingers crossed, that's the solution!
