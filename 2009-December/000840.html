Hey Mike,

Do you remember the bad old days of Xmodem, Ymodem and Ymodem-G ?

How about implementing one of them in MikeOS (Xmodem probably the easiest)
but with a difference....allow uploading of a compiled binary file using Xmodem via the
serial port then being able to actually run the newly uploaded pgm in MikeOS.

This would make development significantly easier and quicker as at the moment it's a case of
write new program, compile add to flp then make an iso then either put on CD or run in a virtual machine.

If MikeOS had this feature, then simply run MikeOS on a dedicated machine and link it to the main
development machine via a null-modem serial cable.

If the newly uploaded pgm to be tested crashes, no problem reboot and start again.
if on the other hand it works a treat, then included it in the flp.

What do you think?
