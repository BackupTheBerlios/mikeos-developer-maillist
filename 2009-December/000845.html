All that is required for Windows is WinImage to modify the flp and make an iso
and goold old Nero if burning to a CD is required.

For a virtual machine, Microsoft's Virtual PC 2004 works well and it's given away free.
Just don't try and use it with Vista.
Vista is not even compatible with Windows!

What I have been doing is slowly porting the MikeOS source code to run directly under Dos (well Dos under XP)
It's rather handy for quick an dirty testing, although XP does not emulate all of the standard BIOS interupts correctly.
For example, there is no support for blinking text and in os_draw_block, you use mov a, ' '.
I had to change this to mov al, 219 as XP won't print spaces if we use colour as well (it just draws  a black block).

For the compiler, I use both Nasm and A86.

Paulo.
