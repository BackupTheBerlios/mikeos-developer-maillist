Hi all,

With Bazza's and Paulo's work, I'm going to create a new area of the
website called Extras. This will contain Takayoshi's C library along
with other projects and files that are useful as add-ons for MikeOS.

Bazza: please email me directly (okachi AT gmail DOT com) with the zip
files. Please include a README in them with some info on what they
do/how to use them and perhaps your email address (so that people
don't ask me about stuff I don't know!) or point them in the direction
of the mailing list if you'll be around here for a bit.

Paulo: as per the other mail before, please let me know if I just need
to add the two in/out_port-type system calls.

I plan to release 4.1 in a couple of weeks, before Christmas. It will
probably include:

* DIR command shows file size info (Gene)
* BASIC interpreter improvements (Steve)
* Another string routine replacement (Ian)
* Better random number generator and RAND BASIC command (me, if I get the time)
* SERIAL commands in BASIC (me, again, given time!)

Plus we'll have the extras from Bazza and Paulo so it'll be a
featureful release. As an aside, there have now been over 1,000
downloads of MikeOS 4.0. Woohoo!

Cheers,
Mike
