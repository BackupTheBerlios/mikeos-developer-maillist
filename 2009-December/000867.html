Hi Mike,

An update on the bin uploader:

It turns out that the xmodem checksum  is quite easy to implement so that is what I've done.
There will be 2 versions available soon:

1) The DOS version which is a mini  HyperTerminal equivalent for computers running without Windows.
2) The MikeOS version.

Both versions are comaptible with  Windows HyperTerminal or any other app that can send files via the serial port
using the Xmodem protocol. 

Below is an example of the xmodem protocol  transferring a bin file of 384 bytes (3 packets):


[RX PC].........................................................[TX PC]
| 
|---------------------\NAK\---------------------------------------->|
|         
|<------/SOH/Blk #1/Blk #1/Good Data/CkSum/------|
|    
|---------------------\ACK\---------------------------------------->|
|     
|<------/SOH/Blk #2/Blk #2/Good Data/CkSum/------|
|  
|---------------------\ACK\---------------------------------------->|
| 
|<------/SOH/Blk #3/Blk #3/Garbled Data/CkSum/---|
|  
|---------------------\NAK\---------------------------------------->|
|    
|<------/SOH/Blk #3/Blk #3/Good Data/CkSum/------|
|    
|---------------------\ACK\---------------------------------------->|
|      
|<--------------------/EOT/------------------------------------------|
|      
|---------------------\ACK\----------------------------------------->|
|                                           
END

All this will be in the help file which I will release at the same time as the uploader source code.
Regards,

Paulo.
