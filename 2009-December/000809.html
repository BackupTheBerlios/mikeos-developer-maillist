Hi Mike,

Will you be adding parallel port access in BASIC as well?
For example:

outb  address, data   (address range  0 - 1023   data range 0 - 255)

outw  address, data   (address range  0 - 1023   data range 0 - 65535)

a = inpb address      (address range  0 - 1023   data returned in
variable "a" 0 - 255)

a = inpw address      (address range  0 - 1023   data returned in
variable "a" 0 - 65535)


And finally if you decide to add the calls to BIOS to find the
parallel port addresses:


a = findlpt n         ("a" will contain address of lpt port "n" or
zero if not found, "n" is 1, 2 or 3)

Wow, a lot of work before you will be able to dig into that Christmas
turkey, but it will add some really nice features to MikeOS.

Paulo.
