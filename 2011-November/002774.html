G'day all,
I released some updates to my programs and the OS.
Memory Manipulator (MEMEDIT.BAS):
	- Fixed the program doing a screen full update after going down
		to the last line.
	- When attempting to save a file that already exists, you are
		asked if you want to overwrite.
	- No longer uses out of range text variables.
BASIC interpreter (BASIC.ASM):
	- Getting the size of an unknown file no-longer causes a syntax
		error.
	- GETKEY can now use arrow key in the same way as waitkey.
	- Size now outputs to 'S' rather than a set variable.
	- Added "PEEKINT" and "POKEINT" commands.
Brain**** Interpreter (BFI.BAS):
	- Updated program for recent OS update.
ASCII Artist:
	- The block cursor no longer turns off after adding a character.
Calculator:
	- Now uses MB++ version 3.0 beta 1, improves menu drawing efficiency.
General:
	- Fixed a bug that causes MikeOS to wipe all connected hard drives :-P
	
The 'basic.asm' is attached my programs can be found on my Google Code:
http://code.google.com/p/mikebasic-applications/

Sorry, about the new commands I added in a feature freeze but my
update is mostly
bug fixes and they will be important.
