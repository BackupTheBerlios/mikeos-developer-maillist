Revamped how many functions work to make them more useful.
It requires the two new commands I added with my BASIC updates, so it
will need beta 8.
Download here:
http://code.google.com/p/mikebasic-applications/downloads/detail?name=mbpp.bas&can=1&q=

Changelog:
- ANCITEXT now uses the current position for the minimum position.
- ANCITEXT now accepts a maximum row in which to terminate at.
- ANCITEXT now uses C = colour; X = maximum column; Y = maximum row
and V = memory location
- XGET/XPUT now handle memory from other segments.
- Added XMEM, must be used before XGET/XPUT and accept V = segment
- XGET now uses parameters X = location; V = retrieved byte
- XPUT now uses parameters X = location; V = byte to save
- There is now only one array.
- ARRAYGET/ARRAYPUT store integers rather than bytes.
- ARRAYGET now uses parameters X = element; V = retrieved integer
- ARRAYPUT now uses parameters X = element; V = integer to save
- New internal variable preservation scheme.

As usual, documentation is available at:
http://mikebasicplusplus.wikispot.org/
