Just a note that BareMetal OS is now able to use all processors in the
computer. Not too shabby for being coded in Assembly!
The system calls aren't quite there yet but the proof of concept works. At
the moment it supports up to 256 processors.

Thanks to Mike for releasing MikeOS and getting me interested in OS
development in Assembly!!

On a side note would it be possible to access the 32-bit memory address
0xfee00000 in MikeOS? If so it would be possible to add mutli-processor
support to it as well :P
