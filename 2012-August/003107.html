>Are there any BASIC commands we could implement that'd make your life
>easier here?

Yes, if you could implement the ability to create complex strings/
I want to be able to create a string with many variable parts,
this normally takes many lines to create.
For example:
$3 = W+" Text "+$4
would be:
34 Text Strings

This would create the string $3 with the string value of W followed by " Text "
followed by the $4 string.This could be accomplished by checking for a '+' at
the end of the line and continuing. I hope this would not be too difficult.

I've attached my progress so far as a binary.
Some of the shortcuts may be different for the unix version.
The binary requires the latest version of MikeOS (4.4b2).

What works currently:
 - Opening a text file via command line (nano foo.txt)
 - Moving cursor around the document, line snapping, scrolling, etc
 - Help function
 - End of line and start of line, cursor position and exit functions

What doesn't work yet:
 - EDITING TEXT!
 - Loading/Saving files inside program.
 - Running BASIC code
 - Selecting/moving text
 - Everything else
