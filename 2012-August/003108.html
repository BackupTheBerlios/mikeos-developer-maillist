Hi folks,

http://mikeos.berlios.de/mikeos-4.4b3.zip

Two big changes from beta 2 here. First, Sudoku is finished (with
three difficulty levels and help), and included. And as per Joshua's
request, the string variable assignment functionality in BASIC has
been greatly expanded. You can now do this, for example:

$1 = "Hello "
$2 = " worlds"
a = 9000
$3 = $1 + a + $2 + "!"
print $3
end

That prints "Hello 9000 worlds!". Joshua: would it help if we bumped
up the strings from 128 bytes to 256 bytes? I'm not sure if that'd
break anything...

Anyway, full changelog since 4.3 below. Please help to test!

Cheers,
Mike
