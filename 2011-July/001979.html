Hi everyone,

http://prdownload.berlios.de/mikeos/mikeos-4.1.2.zip

As discussed, this is exactly the same as 4.1.1, but with the "-O0"
(zero optimisation) option for all NASM calls in the build scripts.
Maybe at some point we'll discover why optimisation is breaking things
in the OS, but in the meantime, this ensures that the assembler does
nothing we don't request, and we're all working from the same base.
