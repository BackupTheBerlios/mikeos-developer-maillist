One of the reasons I think your memory editor isn't working is because Blue
Jay uses a different memory mapping than MikeOS does.
 MikeOS Memory Map:

        0000:8000 +----------------------+
                  | Stack area   (SS)    |
        0000:F000 +----------------------+

        2000:0000 +----------------------+
                  | Kernel code          |
        2000:8000 +----------------------+
                  | User (program) code  |
        2000:FFFF +----------------------

Blue Jay Memory Map:

*        0000:8000 +----------------------+**                  | Stack
area   (SS)    |**        0000:F000 +----------------------+****
 2000:0000 +----------------------+**                  | User code
(CS,DS) |**        2000:8000 +----------------------+**
  | Kernel code  (CS)    |**        2000:FFFF
+----------------------+*
