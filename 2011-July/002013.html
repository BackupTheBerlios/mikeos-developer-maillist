Well, let's clear up some concepts here: MikeOS BASIC is an interpreter; it does not compile anything. And you can't compile MikeOS BASIC programs on Linux either... Not sure what you're saying.

If you mean editing .BAS programs (ie text files) inside a Linux environment, that's totally different -- but yes, that will work if you use a plain text editor. I do all my MikeOS work in Nano, for instance.
