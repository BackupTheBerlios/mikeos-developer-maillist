Hi all,

The version of QEMU in Ubuntu 11.04's repositories does not work well
with MikeOS. File saving operations pause for a long time and then
fail. That's version 0.14.0.

I've just removed that and installed 0.14.1 from upstream source code,
and it works OK, albeit with strange pauses when saving data (it's
instant in VirtualBox). I don't know if it's due to bugs fixed in this
update, or if the Ubuntu packages use a different BIOS image.

Either way, something to keep an eye on.
