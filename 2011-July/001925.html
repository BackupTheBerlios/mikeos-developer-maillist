Here is the updated CLI.ASM I promised. I tested it and it worked under a
Acer Aspire One D255.

Commands added: RN, DEL, NEW, COPY
RN: Rename a file |EX: rn foo.bin hi.bin
DEL: Deletes a file |EX: del foo.bin
NEW: Creates a blank file |EX: new foo.bin
COPY: Copies a file |EX: copy foo.bin hi.bin

Let me know if you find bugs.
