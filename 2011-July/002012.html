Well, let's clear up some concepts here: MikeOS BASIC is an
interpreter; it does not compile anything. And you can't compile

environment, that's totally different -- but yes, that will work if
you use a plain text editor. I do all my MikeOS work in Nano, for
instance.

There are programs available for Linux called unix2dos and dos2unix
which modify text files to suit the OSes. It used to be in a package
called "tofrodos" but I've no idea what it is today :-)
