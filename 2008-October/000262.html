Lo all,

The FAT12 write code is pretty much feature complete now, and can
handle files spread over multiple clusters. I still need to add
error/failure checking and other cleanups. Tomasz has fixed more bugs
in the mouse driver, and I've made a few other changes to the code as
listed below. You can get it from:

http://prdownload.berlios.de/mikeos/mikeos-1.9.3.tar.gz

Full change list:

# FAT12 write support is almost feature complete (multi-cluster write
implemented); writing of files up to 64K is supported; need to
implement error handling and file size info in os_load_file

# os_input_string improved: can now enter and backspace through
strings spanning multiple lines

# PCXVIEW.BIN refuses to display files without a .PCX extension, and
has a softer green background

# TEST.PCX is now in programs/ and copies to floppy disk when
build-linux.sh is run

# More mouse fixes from Tomasz -- works on Toshiba Libretto now

# os_draw_background doesn't clear screen at start, to cut down on flicker

# At kernel start, don't try to set up a block cursor (it's not important)

# build-linux.sh now includes a method for blanking FAT12 floppies at
the top of the code, in the comments
