Actually, your code might still be useful. Even though MikeOS displays
the 'Program Terminated' message, it's still trying to execute a PCX
file which it shouldn't do. It just so happens that whatever data is
in TEST.PCX happens to include a 'ret' early on and ends the program.

machine code, totally messes up the system!
