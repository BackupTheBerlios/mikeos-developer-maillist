I have looked at unreal mode few weeks ago and I think it works pretty the same as a protected mode. We will have to set up elements in GDT (global descrition table). And that all. Then we will be able to use for example:

    mov dx, word [ds:eax]

So i guess we will need define 3 descriptors cs, ds, es.

Using that we'l be able to write a simple memory manager. Actually I am working on it. 
We can use linux alghoritm called "Buddy"

http://www.linuxhq.com/guides/TLK/mm/memory.html

It looks easy to code I guess :)

I have written already a few procedures _mm_alloc, _mm_free, _mm_realloc to manage these 8kB we can use in mikeos (0xE000 - 0xFFFF). It seem working. Then when we will be sure to jump into unreal mode i can rewrite it to manage pages.

When I write enough code I will send a patch.
