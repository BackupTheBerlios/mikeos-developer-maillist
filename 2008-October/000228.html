You could use libmikeos? Take a look the work Takayoshi (is that spelt
right?!) has been doing, as you may be able to adapt it.

If you're dead set on using ASM, then you could set the maximum number of
characters to be stored per line. When this limit is reached (the end of the
screen) or the user presses enter then you can create a new line and you
start again. However, you then need to grab every character in every line
before you can write it to file. This is possible (as far as I can see) in
ASM, but C will allow you to create functions and derived
classes/inheritance a lot more easily.
