I would support switching to PM32-bit for a reasonably good reason. When the
BIOS is phased out and the computers start using UEFI MikeOS would cease to
function. Granted that it will be a while before this happens but it will
happen eventually. Considering that MikeOS is currently using BIOS
interrupts, which UEFI will not support, this seems logical (atleast to me).

I would be glad to hear your views on it.
