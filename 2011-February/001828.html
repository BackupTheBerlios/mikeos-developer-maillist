I use MikeOS Basic for loading .BIN files.

Here's my code:

LOAD "file" 32768     // According to Memorymap from sysdev handbook
CALL 32768            // Run file
LOAD "BFM.BIN" 32768  // Re-load BFM
CALL 32768            // Run BFM

And: When I want to close BFM, system will freeze (black screen), so I jump
back to 0.

CALL 0                // Jump back to MikeOS

This is the best solution that I found. If anyone know better, please share
it.
