> To make the new os call for passwords you can try the following:
>
> Create a new one called os_input_dialog_pwd
> Now copy all the code from the existing os_input_dialog and put it into the
> new os_input_dialog_pwd
> Now change all references to os_input_string to a new one that you create
> called os_input_string_pwd
>
> To create os_input_string_pwd copy all the code from os_input_string then
> change the following in .nobackspace:
>
> ; ======== original ===============
>
> .nobackspace:
>     pusha
>     mov ah, 0Eh            ; Output entered, printable character
>     int 10h
>     popa
>
>     stosb                ; Store character in designated buffer
>     inc cx                ; Characters processed += 1
>     cmp cx, 254            ; Make sure we don't exhaust buffer
>     jae near .done
>
>     jmp near .more
>
> ; ========== change to ==============
>
> .nobackspace:
>     pusha
>     mov al, '*'
>     mov ah, 0Eh            ; Output entered, printable character
>     int 10h
>     popa
>
>     stosb                ; Store character in designated buffer
>     inc cx                ; Characters processed += 1
>     cmp cx, 254            ; Make sure we don't exhaust buffer
>     jae near .done
>
>     jmp near .more
>
>
> I haven't tried it but I suspect it will work or at least get you in the
> right direction.
