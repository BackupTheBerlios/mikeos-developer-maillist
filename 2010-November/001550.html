Raghav,

> Can anyone also tell me how do i jump to label 1 if the strings match?

I assume you are using os_string_compare.
So once you have called os_string_compare, it will return with either the carry 
set ( stings match) or not set ( no match):

jc label1        ; stings match so jump to label1

or if you prefer,

jnc label2        ; strings don't match so jump to label2
