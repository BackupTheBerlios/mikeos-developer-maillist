Hey there everybody,

The first beta of mcc, the MikeOS C Cross-Compiler, is out in platform-independent C source and Win32 binary! It's line-by-line, like old BASIC interpreters, but no line numbers are needed. Just type commands and end your program with return 0; and it will compile, but have these files in the same directory: mcc, nasm, mikedev.inc, stdio.inc. Type #help to get a little help screen with the list of functions/statements implemented so far.

Remember, there's no fixing things in mcc, if you need to change something, you have to do it in temp.asm and then assemble it with nasm by hand. The compiling sequence is as follows: mcc -> temp.asm -> nasm -> mccout.bin    You can then rename mccout.bin to whatever you want to name your program. A lot of user-friendliness things will be in the next release, not as many C-related things.

Download link:
http://willhostforfood.com/users/blasterman/mcc-0.0.1.zip

Email me or post a reply (preferably reply) with questions/comments! Have fun!
