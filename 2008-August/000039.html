Lo all,

Tomasz Gorol has contributed a complete mouse driver for MikeOS.
Here's the patch against 1.4.2 (it's a whopper!):

http://mikeos.berlios.de/mouse/mouse.txt

This implements native system calls, some DOS compatibility, and an
example program. I've uploaded disk images (floppy and CD-ROM) here,
with the build name 1.4.2M:

http://mikeos.berlios.de/mouse/disk_images.zip

So far we've tested it in QEMU and VirtualBox -- I'm going to try some
real hardware tonight. Any other testing muchly appreciated!
