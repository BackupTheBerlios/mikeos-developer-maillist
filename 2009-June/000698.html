Hi all,

I've started work on a BASIC interpreter for MikeOS. So far it
supports these commands:

PRINT LET ALERT INPUT GOTO GOSUB
RETURN CLS CURSOR REM END

Along with variable arithmetic, nested gosubs etc. Numeric variables
are A to Z. See the end of this message for an example MikeOS BASIC
program using these features. To try it out, download the OS from
here:

http://prdownload.berlios.de/mikeos/mikeos-3.4a.zip

Run mikeos.flp as per usual, and go to the command line. Enter 'basic'
to run the BASIC interpreter, which loads TEST.BAS and steps through
it. You can edit the program by entering 'edit', choosing TEST.BAS at
the menu, making your changes and hitting Esc to save.

See programs/basic.asm for the source code -- it's a bit messy and
wasteful in places, but I'll clean it up later. In the meantime, I'd
like to know what BASIC commands I should implement next! I'm going to
add MOVE, SOUND and GETKEY which should facilitate in writing simple
games, and I'll also add some simple string routines too.

Any other ideas for BASIC commands (within reason -- I don't have a
great deal of time to spend on it!)

Cheers,
Mike


REM *** MikeOS BASIC test program ***

ALERT "About to start the BASIC demo..."
CLS

PRINT "Enter a number for variable A:"
INPUT A
PRINT ""
PRINT "Enter a number for variable B:"
INPUT B
PRINT ""

IF A = B THEN PRINT "A is equal to B"
IF A > B THEN PRINT "A is larger than B"
IF A < B THEN PRINT "A is smaller than B"

LET C = A + B

PRINT "A and B added together are..."
PRINT C
PRINT ""

GOTO 50

PRINT "This line will not be executed"

50 GOSUB 100

PRINT "About to end now"
END

100 PRINT "Inside the GOSUB here!"
RETURN
