OK I think I know what you are referring to:

If you run the code on a machine that uses the XT keyboard with the i8048 controller chip, then the INT 9h has to be called or the leds will not switch on.
But these all belong in a museum and  if you have one you should rather contact the "cash in the attic" team.   :-)

All keyboards (as far as I know) since about 1984/1985 are AT types based on the i8042 controller and don't need
the INT 9h and as a bonus also allow the typematic rate to be set.

Later (circa 1987) came the PS/2 standard also based on the same controller chip with enhanced commands from host(pc) to device(kb).
