>Also, the variables sitting at the bottom of both MOUSE driver files, do they need to be setup prior to the file being compiled or is it done during run time?

No setup required, the mouse driver handles them at runtime.
There is a little bit of setup required on the operating system's
behalf however.
Before using the mouse API, the operating system must install the
driver and setup a proper range and scale.
The scale is a value the raw position is shifted by to reduced the
speed of the mouse depending on the resolution/aspect ratio of the
screen. The range is the minimum and maximum positions of the mouse,
this is usually the number of pixels/character on the screen but can
be useful for locking the mouse to a specific area.
TachyonOS uses the following boot code to initialize the mouse for
80x25 text mode.

call os_mouse_setup
mov ax, 0
mov bx, 0
mov cx, 79
mov dx, 24
call os_mouse_range
mov dh, 3
mov dl, 2
call os_mouse_scale
