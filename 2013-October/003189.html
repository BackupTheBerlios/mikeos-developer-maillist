Your code is much better than my early assembly code.

I noticed you use "C-style" loops, like this:

.loop	cmp cl, limit
	je .exit
	; loop body
	inc cl
	jmp .loop
.exit	; other code

when you could be using this:

	mov cl, constant
.loop	; loop body
	dec cl
	jnz .loop

There's nothing wrong with that when your loop limit is a constant, but if  
it is a register, it means you have to dedicate one register to the loop  
counter when you could be using it for something else instead.

Also, the shorter equivalent to CMP register, 0 is to OR, AND, or TEST the  
register with itself. I only mention this because you already use XOR to  
set registers to 0, so you seem to like this kind of tricks. :)

The game is pretty good and it makes really good use of CP437 characters,  
especially the hook. I think the fish should be white to make them easier  
to see, or of different colors for some variety. When you catch a fish,  
maybe it could reappear at a different edge and direction (chosen  
randomly) so their movement isn't so predictable.
