Hey guys,
I've created a mouse library for MikeOS by modifying the TachyonOS
mouse code. This library can be included with any assembly program to
give it mouse capabilities.

Functions:
mouselib_install_driver --- Installs the mouse driver (use this first).
mouselib_range --- Sets the movement range of the mouse.
mouselib_scale --- Sets the movement speed of the mouse.
mouselib_remove_driver --- Removes the mouse driver (use this at the end).
mouselib_locate --- Returns the current mouse position.
mouselib_move --- Sets the mouse position.
mouselib_show --- Shows the mouse cursor in text mode.
mouselib_hide --- Hides the mouse cursor.
mouselib_wait --- Waits for mouse input.
mouselib_anyclick --- Checks for any mouse button currently pressed.
mouselib_[left/right/middle]click --- Checks for a specific mouse
button currently pressed.
mouselib_input_wait --- Waits for either mouse or keyboard input.
mouselib_freemove --- Allows the user to move the cursor and returns
the position if a key is pressed or mouse button clicked.

You can get the library from the downloads section of my Google Code.
https://code.google.com/p/mikebasic-applications/downloads/detail?name=libmouse-1.0.0.zip&can=2&q=
