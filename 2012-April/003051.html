Okay, here it is:
http://code.google.com/p/mikebasic-applications/downloads/detail?name=mbpp.bas&can=2&q=

I created an alternate version without the new features but with the bug fixes.
This is a large clean up for the library, making the library more
compact and efficient.

ANCITEXT is now be potential hundreds of times faster because it uses
a DO/LOOP which jumps
instead of a GOTO loop that must search the program. Also the inner
loop has been optimized.

All the library's functions now preserve your variables. I patched up
the few remaining that would
destroy the user's variables. Except for 'STARTPRG' because of program
flow issues.

The 'box open' function no-longer replaces the users error message
with it's own unless there is
actual an error. If a string is too long it now identifies the problem
string rather than a single
message for all.

I also cleaned out the rubbish by removing all the functions that were
unused, broken, unfinished
or obsolete. I made a decent space saving out of this update making
this version 1.4 KB small
than the previous version with all the same features, that must be a
new personal best.
