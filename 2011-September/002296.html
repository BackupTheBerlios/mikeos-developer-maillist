By the way, a comment in MEMEDIT isn't quite accurate:

  "rem progstart is where the program is loaded to. While this is always 32767
  rem on MikeOS some OS's that run MikeBASIC are different."

The number is 32768, and it isn't ALWAYS that. It will be 32768 if the
program is launched directly by the kernel via the program menu or
CLI. However, if it's launched by another program, PROGSTART will be
different. For instance, if it's run via EDIT then the BASIC program
is loaded 4K after EDIT -- ie 4K after 32768.
