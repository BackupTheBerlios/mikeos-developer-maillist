On Tue, Sep 27, 2011 at 9:23 AM, Joshua Beck <mikeosdeveloper at gmail.com> wrote:
>
> How about multiple conditions for if statements?
> This would save a lot of coding
> ie Changing lowercase to capital
> Would be this:
> IF X < 96 AND X > 123 THEN X = X - 32

You can do that at the moment like this:

IF X > 96 THEN IF X < 123 THEN X = X - 32

(For converting to lowercase you need values between 97 and 122, not outside.)

The parser just treats anything after THEN as a new instruction, so
you can put another IF; if any IF condition isn't true, the parser
skips to the next line. However, to keep MikeBASIC in line with other
dialects I might add AND for 4.3.
