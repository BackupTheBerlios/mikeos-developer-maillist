Hey all,

I'm posting to release the first version of the MikeOS Extended Text 
Editor! The editor, which has the same filename as the default one, is 
based off of the default text editor and has a few useful new features, 
such as the ability to create new text files and to change the filename 
in the editor (useful as a proto-Save As).

A complete changelog is at the top of edit.asm included in the attached 
ZIP file.

Please leave suggestions and any bugs you find - I'd love to hear about 
the former and squash the latter!
