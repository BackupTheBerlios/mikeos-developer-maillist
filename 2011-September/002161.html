And now I've just implemented parameter passing, more along Matej's
line of thinking: the program receives a string in SI with all the
parameters excluding the program name, so it can do as it pleases.

The reason is, I've implemented it in the BASIC interpreter too! On
startup of a BASIC program, $1 contains the parameter string passed,
or it's empty otherwise. So if you have this BASIC program:

print "The CLI passed us:"
print $1
end

And then run it from the CLI like this:

foo.bas Some words here

Then it will print "Some words here". Hopefully that'll be useful to
Bazza and Joshua :-)
