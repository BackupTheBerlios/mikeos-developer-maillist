Well, I downloaded MikeOS 4.2 beta 2 and it has
version 1.0.0 of Calculator which uses MB++ 2.1.0.
This has a bug that causes the ending animation to
be displayed in the colour black even in no interface
mode. This causes a few seconds "black screen" before
the BASIC finished message appears.
This was fixed in MB++ 2.1.5, which Calculator version
1.0.2 uses. Try downloading the latest version of
Calculator and see if the bug still happens.
