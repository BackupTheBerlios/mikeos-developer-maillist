On Sun, Sep 18, 2011 at 3:14 PM, Matej Horvat <redjazz_slo at yahoo.com> wrote:
>
> But what string manipulation can BASIC do anyway? It would be nice to have a SADD command to get the address of a string. Then you could use PEEK and POKE to work with characters, if that's not considered too low-level for BASIC.

You can do that already! It's all explained in the Handbook:

http://mikeos.berlios.de/handbook-appdev-basic.html

"Arrays -- You can use string variables as arrays via the PEEK and
POKE commands. For instance, X = & $1 places the memory location of
the $1 string variable into X. You can then put data into it with eg
POKE 77 X (put 77 into the memory location pointed to by X)."
