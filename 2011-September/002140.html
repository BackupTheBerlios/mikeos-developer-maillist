OK, I've had more time to play around with the new shell and editor
now. My thoughts:

1) Good work Troy! Well formatted and commented code, and no bugs that
I can see, but I'd like to give it lots of testing.

2) I'd like to simplify the text editor so that instead of Rename we
have Save As. So you hit F2 to save with the current name, and F3 to
save with a different. Put New in F4. You can add space for the Save
As help in the bottom info bar by putting "Press Esc to quit" in the
top bar.

3) Let's make the default new filename "UNTITLED.TXT" rather than just
"UNTITLED". MikeOS requires that filenames have three-letter
extensions -- when it's "UNTITLED" it doesn't save correctly (I'll
update the error messages to show that)

4) When you use "del", it shows the filename in the resulting text:
"Deleted FOO.BIN". That's cool -- can we do the same for "copy" and
"ren" etc.? Eg "Copied BLAH.FOO to LOLWUT.BAR" as the output text?

5) There are a couple of things in the code that could be clearer.
Line 27 of your cli.asm has this comment: "The final prompt is
returned in AX", but that makes no sense to me there (presumably it's
left over from some other code). Line 323 has an instruction commented
out -- can we just get rid of it? Same with line 60 of edit.asm. See
if there are any other comments you can add to explain things.

If you can fix those and re-send the code, that'd be great! Otherwise
it looks ready to go into 4.2 -- nice work. Thanks!

M
