Version 0.4.6 of BareMetal OS is out now.
http://www.returninfinity.com/baremetal.html

BareMetal OS started out as a 64-bit version of MikeOS.

New in this version is the ability to write to a FAT16 formatted HDD.
All the code is in assembly and should be easy enough to port back to
16-bit if someone is interested enough.

FAT16 code is here:
http://code.google.com/p/baremetal/source/browse/trunk/os/drivers/fat16.asm
HDD code is here:
http://code.google.com/p/baremetal/source/browse/trunk/os/drivers/hdd.asm
