I created my own MikeOS-based operating system in order to combine all
the new code
and have an easy way for everyone to run all my BASIC programs.

Stuff I've changed:
 - AAP file as background
 - New main menu system
 - New UI colour theme centralized in the kernel
 - Extended menu shell code (by Robert Looney)
 - MikeOS File Manager replaced with BlackSVK File Manager.
 - MikeOS Editor replaced by Yotta Text Editor
 - Latest BASIC interpreter used
 - Minor API changes
 - Heaps more programs included

I've uploaded the code to a Dropbox account, I will probably start a
Google Code site
or something like that else eventually.

There are probably a few licence issues because I've GPL'd it but
there's BSD'd and
proprietary software included and there is more documentation I could
do but the
system works and it's really stable. Give it a try!

Thanks to all the MikeOS contributers that have helped make the code I
used in this
operating system.
