>So I am introducing what I am dubbing as the MikeOS Extended Shell.
>The shell is just a very simple addition to MikeOS. After you select a
>file it shows the new screen (screenshot attached). The functions are
>quite simple and to be expected, Execute, Delete, Rename, Show file
>size, (and a couple of additions i have added to the OS) Restart, &
>Shutdown.

That's awesome! I like how it gives you a menu for each application,
you just need to add copy and create and FILEMAN would be obsolete.

I think we need a main menu for the tasks not related to the actual file,
TUI, CLI, Reboot and Shutdown and maybe other stuff would be on
this menu.

I added the ability to set an ASCII Artist picture as the 'background'
to the shell. The picture must be create with ASCII Artist 3.0+, with
the dimensions 76x21 (default size) and be saved as
'BACKGRND.AAP'.
