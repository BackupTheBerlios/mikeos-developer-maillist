G'day all,
I've updated Memory Manipulator and ASCII Artist to
accept command line parameters. Parameters are not case sensitive.

ASCII Artist simply uses it to load files, ie "draw foo.bar".
I think it saves a bit of time over scrolling the file list,
after all, everything is more efficient by command line.

Memory Manipulator is a little more complex. It processes all
input into different parameters which are all seperate. There are a number
of parameters:
"help" prints all comments
"nosnd", which prevents the startup sound.
"noinfo" prevents the whole startup screen.
"dblborder" uses double instead of single box drawing characters for the
border.
"sldborder" simply uses block characters for a thick solid line.
"red" unlocks a badass new colour scheme.
You can only use three parameters at once.
I'd appreciate testing.

Joshua Beck, http://code.google.com/p/mikebasic-applications
