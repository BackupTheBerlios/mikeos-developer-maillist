Hi all,

My HW debug kernel has paid dividends.
I am now booting MikeOS from a USB flash
formatted to FAT16.

I used a modified MikeOS kernel (from Mike)
which bypasses looking for autorun.bin and .bas
(for obvious reasons) and I get the welcome GUI
screen, I then select CANCEL go to CLI and can
even run any command there that does not involve
reading or writing a file from/to disk as the kernel
routines are still for FAT12 drives.

I will document exactly what I did and post it shortly.
There is still a lot of work to be done, but at least
the first step has been taken.

Paulo.
