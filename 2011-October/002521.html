I ran into a few problems with the game.

 

1) To compile the program into the MikeOS disk image you should rename in to
adventure.bas instead of ADVNTURE.BAS. The build script (on OS X at least)
is case sensitive when it loops through the programs folder to find .bas
files.

2) When I run the program the initial storyline PRINTs work fine after that
when the program reaches the second input (the first being the name) nothing
happens. I entered 1, 2, 3, 4, 5, 6, 7, 8, 9, open and Open. Nothing
happens. It doesn't go to the rooms.
