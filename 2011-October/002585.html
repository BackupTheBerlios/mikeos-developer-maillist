G'day all,
Quite happy with this application.
It's an interactive MikeBASIC interpreter.
No longer do we have to create blank for use in editor if you just want to
use a single command, or experiment with new commands or just mess around.
It's written in BASIC and it runs the command with self-modifying code (yay).
The environment is not affect by any throw away variables by a bit of
peek'n'poke.
You can use ":" to separate commands,
i.e. "FOR X = 0 TO 15 : INK X : PRINT "Using Colour: " : PRINT X : NEXT X".
It starts in interactive mode if you load it for the menu or with no parameters
but you can use it in the commandline like: "BASIC LOAD FOO.BAR :
PRINT "FOO.BAR loaded!""
You can get it off my Google Code from:
http://code.google.com/p/mikebasic-applications/downloads/detail?name=BASIC.BAS&can=2&q=

By the way there's a small bug fix for GETBYTES that fixes a bug in
which location over
32k don't work properly.
http://code.google.com/p/mikebasic-applications/downloads/detail?name=GETBYTES.BAS&can=2&q=
