G'day all,
I've made a new expanded version of my interpreter the includes the
MB++ library (is it a library?). It prompts for a command in a MB++ input box
and displays text in a message box. It can be used to simulate the framework.
It has a preconfigured start section with a default configuration.
It starts with no interface but you could get the interface up by doing:
  $z = "BASEINT":Z = 14:GOSUB REFRESH
  $t = "Application" : GOSUB SETTITLE
In this version you will have to move the cursor to print text however,
i.e. 'MOVE 10 2 : PRINT "Hai World!"'.
All text strings are preserved between commands.
This version will be separate from the normal version.
Get it from my (well, not just mine anymore) Google Code at:
http://code.google.com/p/mikebasic-applications/downloads/detail?name=BASIC2.BAS&can=2&q=

PS I'm working on a Brain**** interpreter.
http://tinyurl.com/4f6mt
It will only be an 8-bit version, meaning 256 bytes of RAM
maximum but the variables are set at the start and you can change that.
I haven't quite got those bracket figured yet, it seems to be
jumping to weird addresses.
