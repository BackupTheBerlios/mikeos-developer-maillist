------------------------
 bootdev values
------------------------
  00h = 1st stiffy
  01h = 2nd stiffy
  80h = 1st HDD
  81h = 2nd HDD

Then at the beginning of the kernel
before call os_seed_random

mov [bootdev], dl

then optionally you could add two bytes
to describe the boot drive if used in other pgms

assign_letters:

cmp dl, 00
je is_a_drive
cmp dl, 01
je is_b_drive
cmp dl, 0x80
je is_c_drive
cmp dl, 0x81
je is_d_drive

mov [DriveLetter], '?'
jmp carry_on_with_kernel

is_a_drive:

mov [DriveLetter], 'A'
jmp carry_on_with_kernel

is_b_drive:

mov [DriveLetter], 'B'
jmp carry_on_with_kernel

is_c_drive:

mov [DriveLetter], 'C'
jmp carry_on_with_kernel

is_d_drive:

mov [DriveLetter], 'D'
jmp carry_on_with_kernel

The variable DriveLetter could then
be used in other routines to say confirm
which drive to save or load a file from/to.
