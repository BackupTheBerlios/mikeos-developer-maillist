I added another couple of commands the BASIC interpreter.

---Get cursor colour---
i.e. CURSCOL A
Gets the colour of the character below the cursor.
I didn't want to add it to CURSCHAR because the rewriting
would be enormous.

---Get timer value---
i.e. T = TIMER
Get the lower word value of the system clock.
Useful for performance testing and checking for a delay without
stopping the program.
Added as a keyword.

---Change the page---
i.e. PAGE 1 0
Sets the working page and the active (display) page.
Useful for smooth animations / screen redraws.
The command was easy to write but getting all the command to display
on the working page was a nightmare. INPUT still uses active page.
Remember there will be a delay between switching the display page
and it appearing on the screen.
