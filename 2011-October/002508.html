G'day all,
Just another small application.
a password protection system for MikeOS.
The zip file contain two BASIC programs.
AUTORUN.BAS and NEWPASS.BAS

NEWPASS.BAS prompts for a new password, encrypts it (very basic) and
save it to PASSWORD.SAV.
AUTORUN.BAS loads and decrypts the password file then asks the user
for a password.
If the password is incorrect, it waits a few secs then beeps a low frequency.
If the password is correct it beeps a high frequency and ends the
program (continuing the boot).
Several attempts will freeze it up. If there is no password file then
the program just ends.
You can get it from:
http://code.google.com/p/mikebasic-applications/downloads/detail?name=PASSWORD.zip&can=2&q=
