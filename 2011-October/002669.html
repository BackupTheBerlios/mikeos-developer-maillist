It says this CAN happen. There's no guarantee that's the case, and
everything I've read about it is horribly vague. With the BIOS, we
know we always have interrupts for reading the keyboard and sectors
from a drive. With UEFI, it seems we'd have to check if certain
features are provided, which they might not be on some machines, and
it gets very messy very quickly.
