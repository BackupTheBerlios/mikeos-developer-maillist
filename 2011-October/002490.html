Hi Joshua,

Nice apps.
I quite like the CLI memory viewer idea.
You should compile them as bin files and it
will make them even more useful.
For example with head and tail you could add
an extra command line parameter such as '>'
which would mean redirect output to a file
as follows:

tail myfile.bas 20 > myfiletoo.txt

The last 20 bytes of myfile.bas would be
written to myfiletoo.txt
If myfiletoo.txt does not exist it will create it.
You could even go one step further and add
'>>' which would mean append to a file.

and the same for getbytes.

Paulo.
