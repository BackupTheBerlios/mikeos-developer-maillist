-> Added the ability for FOR/NEXT loops to include variables.
FOR X = A TO B
I have found myself needing this multiple times, which I have
to do a LABEL/GOTO loop.

-> Added beep command.
BEEP
Just a simple sound for those who just want a standard beep.

-> Added number to string and string to number converter.
NUMBER $1 X
NUMBER X $1

-> Added size command.
SIZE $1
SIZE "FOO.BAR"
Get the size of a file before you load it. You can make sure
that you have enough space for the file.

-> Added rename command.
RENAME $1 $2
RENAME $1 "FOO.VAR"
RENAME "FOO.BAR" "FOO.VAR"
The missing file command. You can use strings or quotes.


-> Fixed my PRINT CHR bug
The cursor will now move for "PRINT CHR X ;" except if it's
at 79,24 then it will stay. New lines will still be printed
without the semi-colon.
Should fix Muncher and make full screen borders possible.

instead of capitalize 8.3 file-names without extensions?
