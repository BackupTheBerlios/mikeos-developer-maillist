Hi all,

I'm making progress with the FAT16 bootloader.
I could not understand why the bootloader would
load the kernel but it would not display properly
and would hang after a key press.

So I made a simple "hardware debugging" kernel
that sends the data in the segment regs, stack pointer, etc
via the serial port to another machine for further investigation.

It has already been a great help, look at the crazy values
in these regs shown in the attached picture.
