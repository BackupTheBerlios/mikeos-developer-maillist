Now, if anyone wants to draw lines from arbitrary positions, look into
Bresenham's line algorithm. The Wikipedia page has this pseudocode:

--- CODE ---
plot(x0,y0, x1,y1)
  dx=x1-x0
  dy=y1-y0

  D = 2*dy - dx
  plot(x0,y0)
  y=y0

  for x from x0+1 to x1
    if D > 0
      y = y+1
      plot(x,y)
      D = D + (2*dy-2*dx)
    else
      plot(x,y)
      D = D + (2*dy)
--- CODE ---

Someone should be able to adapt that to a MikeBASIC routine that
programs can GOSUB to! The MikeBASIC version will be more verbose as
we don't have brackets in arithmetic operations etc., so such
operations would have to be done in more steps.

An internet high-five to the first person to do this in MikeBASIC!
Full description of the algorithm here:

http://en.wikipedia.org/wiki/Bresenham's_line_algorithm
