BASIC will freeze up with a syntax error if you try
to read past the end of the data section, i.e.
FOR X = 1 TO 4
  READ DATA Y X
NEXT X
END
DATA:
20 40 80

This will clear the screen and print the text
"Error: syntax error", which is unrecoverable
because there is no way to exit it.
