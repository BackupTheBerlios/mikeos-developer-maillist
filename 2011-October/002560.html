This is correct. Actually, it's drives smaller than 1GB which includes 512 
MB.
However these are older machines, which are still in use.
These BIOS/versions will only boot and run a FAT12 floppy disk image.
The important thing is that the other BIOS/versions which only boot and run
as HDD will boot and run the same FAT12 floppy disk image.
