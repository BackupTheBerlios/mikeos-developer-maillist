G'day all,
I made some new command line applications "head" and "tail".
You can use them like:
head (filename) [lines]
tail (filename) [lines]
You can only read text files, so will only accept .TXT and .BAS files.

Here are some examples that will work:
head memedit.bas
head tail.bas 12
tail draw.bas 8
tail readme.txt 20
tail head.bas

Here are some example that will not work:
head kernel.bin      <-----Unsupported extention
tail calc.bas 200    <-----Too many lines
head asdfsdasdfga.txt <----Invalid filename
head *#&$%@#vbCrLf%%@.wtf 500000 <---All of the above
tail foo 10          <-----No extention

Also I made a kind of command line memory viewer "GETBYTES.BAS"
Simply type "getbytes start end" and it will be neatly printed to the
screen in hexdecimal. For example, "getbytes 100 200".

Available on my Google Code at http://code.google.com/p/mikebasic-applications
