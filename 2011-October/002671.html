The problem with PM32 is you no longer have access
to BIOS which means using drivers.
The problem with drivers in that there are no
"one size fits all" and individual drivers will be
needed for sound cards, vga, ethernet cards,
the list continues and of course  there are thousands
of different motherboards.

Infact this would be such a monumental undertaking
that it would be easier just to use Linux customised.
