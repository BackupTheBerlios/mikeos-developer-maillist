On Mon, 3 Oct 2011 20:42:38 -0500, Justin Tokarchuk wrote:
> is 33h reserved for DOS? Do I have to write my own driver?
Yup, int 33h is a DOS mouse driver interrupt. All you have available is 
the keyboard controller and/or int 15h AX=C200h through C209h. These 
interrupt functions allow you to initialize the PS/2 mouse port, 
configure it, and do read/write operations on it.

A good-looking resource for the PS/2 mouse protocol can be found here: 
http://www.computer-engineering.org/ps2mouse/
