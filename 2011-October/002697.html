G'day all,
I recently added video pages to MikeBASIC.
This gives us the option to draw to one page while showing
the other.
How does this help? Smooth animations.
I created a demo application to show the difference.
You use the WASD keys to move a text sprite around the screen.
You can use Q and E to control the speed by default it will
run on the fastest speed using two video pages.
You can press T to toggle between using the same page and
using two pages. The page is displayed at the top.
