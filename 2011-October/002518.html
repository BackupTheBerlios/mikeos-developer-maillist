It depends. I have never used the fetch command personally.

On my system I have a folder called GitHubMain in which I have individual
folders for each of my GitHub repositories. When I need to update it with
the online one I cd to the respective folder and then type "git pull origin
master". This will update your copy of the repository with the latest code.
You will have to replace origin with the name with which you added the repo.
If you added it using GitHub's default instructions it will be origin.
Replace master with the branch you want to pull. In the case of
MikeOS-Projects the branch will stay as master.
