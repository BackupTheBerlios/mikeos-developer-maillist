Nope - 1000o is 200h, 1000000000b or 512 (decimal).

A very good link for these calculations:
http://www.microcontroller.com/Embedded.asp?did=92
However, this site uses the prefix notation rather than the suffix one I'm
advocating.

It doesn't matter which we use, so long as we're consistent. I personally
think it's conceptually easier for new users to work with suffixes (the
first thing I did when confronted with MikeOS is alter the notation in this
way so I could better understand it!).
