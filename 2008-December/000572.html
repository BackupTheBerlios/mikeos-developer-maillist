Hi,

Here is the test version of libmikeos for MikeOS 2.9.x.
At least "Hello, world" works, but others are not checked.

http://www.bi.a.u-tokyo.ac.jp/~uaa/gomitext/2008/20081213/20081213.tar.gz


The size of stack is configurable.
To tell the address of stack area, please use ld86's -H (heap address)
option.

All samples use 0xd800, so memory layout is :-

	0x8000	+-------------------+
		| Code/Data (CS/DS) | (22k)
	0xd800	+-------------------+
		| Stack (SS)        | (6k)
	0xf000	+-------------------+
