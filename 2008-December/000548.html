Back on topic, it looks like suffixes is the way to go: FAh,
11001001b, 1234o (if we ever need it!).

Are there any other aspects of consistency we should look out for in a
tutorial OS? I've replaced all instances of 'xor reg, reg' with 'mov
reg, 0' for clarity -- I know it's slower, but it's easier to grasp
for those new to binary arithmetic.

If there's anything anyone has noticed in the code that's not
clear/consistent, please let me know!
