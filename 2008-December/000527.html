Hi all,

This is 'feature complete' now -- I don't plan to add any new system
calls before 3.0. All of the system calls are now split up into their
relevant files, and I've started to clean up the code with better
comments and tweaks.

Download: http://prdownload.berlios.de/mikeos/mikeos-2.9.4.zip

Changes:

# All system calls now separated in source/features/

# In program selector at start, Esc now returns to the GUI/CLI choice dialog box

# Started updating kernel code: new comments, and instances of 8
spaces changed to tabs

# Updated bootloader: use 'mov ax, 0' instead of 'xor ax, ax'; and
'mov si, foo' instead of 'lea si, [foo]' for consistency and
simplicity

# More video fixes to (hopefully!) prevent flashing text on real hardware
