I've created two new string command that make it easier to transfer
string between
string variables and memory, which I do a lot.

Here's the syntax:
STRING [store/load] [string variable] [number/numerical variable]
STRING STORE $5 60000
STRING LOAD $8 W

I added them onto basic.asm. New code and diff log is attached from
4.3b4 is attached.
