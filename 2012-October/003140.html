G'day all,
I did a complete rewrite of ASCII Artist to support a new file
format with a title, colour, custom size and keymaps.
There have been load of commands added to the menu and
menus now contain five item and exit with escape rather than
four items and a back button.
There is also an up-converter for the old file format.

Now requires the MB++ library version 3.2.2
http://code.google.com/p/mikebasic-applications/downloads/detail?name=libMB%2B%2B-3.2.2.tar.gz
and the basic.asm I just submitted.

It will work with MB++ version 3.2.1, as included with the beta
builds of MikeOS but it will be a little bit buggy.

Get it here:
http://code.google.com/p/mikebasic-applications/downloads/detail?name=asciiartist-3.0.0.tar.gz
