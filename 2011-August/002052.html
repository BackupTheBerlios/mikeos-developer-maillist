Finished work on APPDEV 2.0, a zip file is attached.
I'll be using this Gmail Account from now on.
I've added:
XGET/XPUT - 8k memory extension.
NUMBOX - Box displaying numbers.
DINBOX - Box with two inputs.
STARTPRG - Intelligent program starter.
LOADBOX - Scroll-able file list to load files.
MENUBOX - Menu with five options.

Information is available in the manual, which I have extended the
information on all functions.
I added in a cool calculator application that shows the new functions in a
practical program.
Enjoy guys!
