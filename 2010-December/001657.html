Hi Mike...

> I want to include either Function 0x2 or Function 0x12 to the keyboard.asm,
> KB Flags and shift them IMMEDIATELY from a tmp store to a known address.
> 
> I decided on 8 Bytes so that the possibility of 4 Words, 2 Double Words, (Long),
> and 1 64 Bit quantity, (Long Long?), could be stored.
> 
> We can leave function(s) as undocumented so that these locations can be
> allocated by any other programs if the undocumented functions are not used.
> 
> Are you game for this none standard idea?





Code add_on for keyboard.asm:-

; ------------------------------------------------------------------
; os_key_flags -- Looks at address 0x0000:0x0417 for the KB flags
; IN: Nothing; OUT: AL = ?? stored directly at user address 65528
; Leave this as undocumented for the time being!

os_key_flags:
	pusha

	mov ax, 0
	mov ah, 2			; BIOS KB flags call
	int 16h

	mov di, 65528			; Point to address 65528
	mov byte [di], al		; Store the byte for reading

	popa
	ret
