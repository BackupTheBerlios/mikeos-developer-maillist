But anyway, in the link above (and below) is modified cli.asm code
that allows command line arguments.  And an echo program is possible
because of that now, too.

http://mcosorg.x10.mx/cli.asm

The programs receiving the arguments only have to mov 'em, like so:

start:
     mov si, bx
     call os_print_string
     call os_print_newline

     ret

used on the command line like
> program.bin arg2 arg3 arg4
