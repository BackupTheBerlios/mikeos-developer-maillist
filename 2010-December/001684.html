2010/12/20, M Saunders <okachi at gmail.com>:
> Hi folks,
>
> It's here: http://prdownload.berlios.de/mikeos/mikeos-4.1.1-test2.zip
>
>
> Changes since 4.1.1-test1 are:
>
> # source/features/screen.asm: os_list_dialog now scrollable for more
> entries than can fit into the dialog box
>
> # source/features/cli.asm: dirlist expanded from 255 to 1024 bytes
>
> # source/features/screen.asm: os_file_selector: buffer expanded to 1024
> bytes
>
> So the main change is that the dialog box is now scrollable. I've
> tried this out by putting 25 files on the floppy disk and the menu
> scrolls (using the cursor keys) through them. Please test and report
> any bugs!
>
>
> And the changes since 4.1 (for those just joining in) are:
>
> # Build scripts now copy all .bas files in programs/ to the floppy
> disk image (and not just example.bas)
>
> # source/features/misc.asm: os_pause rewritten by Paulo Valongo to
> work with more BIOSes; now takes 110ms chunks in AX
>
> # source/features/disk.asm fixes from Mike Gonta: os_get_file_list now
> checks for directory entries AND volume label; os_write_file routine
> now properly zeroes out the .free_clusters list
>
> # Documentation and comment fixes
>
>
> That should be everything for 4.1.1 now. I'll leave it up over
> Christmas, and if nobody finds any problems, we can release it as
> 4.1.1 final in the new year. Thanks in advance for any testing and
> feedback!
>
> Cheers,
