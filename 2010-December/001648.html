I did some simple graphics a little while ago and I used this to get me into
graphics mode:

*mov ax, 13h   ; AH=0 AL=13, MCGA 320x200 256 color graphics mode
int 10h

*And to plot a pixel I used this to plot pixels:

*mov cx, 320 ; Width of the screen
mul cx
add ax, bx

xchg si, ax
mov bx, 0a000h
mov es, bx
mov word [es:si], 11 ; display bright pixel*

AX is the column (Y coordinate) and BX is the row (X coordinate)... I
think...  :)
