Hi all...

Here is the code for the keyflags routines.

It gives real time detection of Left Shift, Right Shift, Ctrl and Alt keys in the lower
4 bits.

Also Caps Lock, Num Lock, Scroll Lock (and Insert) toggled state in the upper 4 bits.

In the archive, (inside a KEYFLAGS.DIR drawer), is basic.asm, keyboard.asm and
a BASIC test code KFBAS.BAS for testing inside MikeBASIC.

This can be used as a trigger for special actions in any code you write but in my
case for the KB I/O project as it makes this now purely MikeBASIC only.

The OS call is os_key_flags and the BASIC call is KEYFLAGS x, (or any letter of your
choice).

I will write exactly how to use it after XMAS but ATM here is the archive...
