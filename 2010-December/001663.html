The MikeBASIC code to test the os_key_flags function. 
I called it INT16-02.BAS 
 
 
 
CLS 
 
REM Call the kernel offset vector, 0x00CF, 
REM 207 decimal, for the "os_key_flags" subroutine 
REM added to kernel.asm and recompiled. 
loopit: 
CALL 207 
 
REM Read the known address for the KB Flags. 
PEEK a 65528 
 
PRINT a ; 
PRINT " " ; 
 
REM A getout clause if Scroll Lock is ON only. 
IF a = 16 THEN GOTO getout 
PAUSE 1 
GOTO loopit: 
 
REM Clean exit. 
getout: 
CLS 
END
