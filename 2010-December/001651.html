Hi Derek...

> Already knew how to switch into graphics mode, but how can I actually plot
> pixels to the screen?

We will assume EGA lores here, 320 x 200 x 16 colours.
Palette is fixed exactly the same as CGA.

DEMO Write pixel.

mov ax,0dh ;Video Mode 13 decimal, 320 x 200 x 4 bit EGA lores display.
int 10h ;Do it.
;Write a single pixel to this window.
mov ah,0ch ;Pixel Write function.
mov al,0fh ;Bright white, IIRC, value.
mov bh,0h ;Start with PAGE NUMBER 0.
mov cx,0h ;Top Left Hand Pixel Column.
mov dx,0h ;Top Left Hand Pixel Row.
int 10h ;Plot it.
ret
;Top Left Hand Corner should have a white dot.

Register al can hav 0 to 15 decimal.
Register bh can have 0 to 7.
Register cx can have 0 to 319 in THIS MODE.
Register dx can have 0 20 199 in this MODE.

Fully BIOS controlled.

DMA method is possible in some modes but walk first.
