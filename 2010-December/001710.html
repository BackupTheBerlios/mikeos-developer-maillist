Hi! I thank you all for helping me implementing system calls in my OS. MikeOS kernel format is like this...

jmp os_main                                 ;0000h       
jmp os_print_string                        ;0003h
...

the mikedev.inc

os_main                                 equ 0000h       

os_print_string                        equ 0003h

I've tried several combinations and mine is 

kernel.asm

jmp System.Initialize                                ;0000h            
jmp System.IO.Screen.DisplayString          ;0002h
...

dev.inc
jmp System.Initialize                                equ 0000h            

jmp System.IO.Screen.DisplayString          equ 0002h

Now i can implement other system calls and it works. Thanking you all again for your precious help.
