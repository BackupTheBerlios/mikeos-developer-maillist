G'day all,
I released some updates for a few of my main applications:

###MB++###
Release version 3.0 (finally), no more beta's.
Changes since 3.0b3:
 * Making an error message too long now causes a fatal error.
 * Fixed border drawing.
 * 'SETTITLE' once again prints the title, for compatibility.
 * 'SETTITLE' will now fill the whole title line.
 * 'SETTITLE' does not run if the length of $T is not between 1-78 characters.
 * 'OPENBOX' now checks the length of all the text strings before use.
 * Added a method of preserving and restoring cursor position and ink
colour (SAVELOC/LOADLOC), used in TITLE, SETTITLE, OPENBOX, CLOSEBOX
 * Menu's will not highlight starting indent.
 * Fixed 'SAVEVAR' and 'LOADVAR' functions getting the wrong numbers.
 * Errors 'SAVEVAR' and 'LOADVAR' are now crash the program ("FATAL:").
 * Fixed 'TITLE' and 'SETTITLE' not printing.
 * 'SETTITLE' no longer accepts a colour, uses border colour instead.
 * 'BORDER' now preserves your variables, cursor position and ink colour.
 * Cleaned up 'MENUBOX'.
 * Some work on 'LOADBOX'.
 * 'CONTENT' sub is now optional.
 * New implimentation of 'STARTPRG'.
 * Fixed inbalance in 'SAVEVAR' and 'LOADVAR' saved location.


###ASCII Artist###
version 2.2.1
 * Rewritten to use MB++ version 3.0

###Calculator###
