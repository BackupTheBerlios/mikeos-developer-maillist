Hello Blake,

Interrupts are very simple(in real mode). At the first 256 memory
locations,there are 2 byte segment-offset vectors which point to the
interrupt handlers.Just address the vector you want by using the
formula (interruptno * 2).Interrupt handlers are just like
procedures,but instead of a ret,use an iret.Also,you got to save the
registers(not necessary,but a good convention).Note that this only
works in real mode.
