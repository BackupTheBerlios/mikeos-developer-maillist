Hello,

I am trying to make MikeOS binaries with Bruce Evans' bcc compiler,
used for LILO/Linux 8086.

Here is current source package, work in progress.

http://www.bi.a.u-tokyo.ac.jp/~uaa/gomitext/2008/20080915/20080915.tar.gz

Requirements:
	Robert de Bath's Dev86-0.16.17 (obtain from http://www.debath.co.uk/)
	GNU Make (not BSD make)
	386+ processor

Supported function:
	Almost all MikeOS API
	bcc basic 32bit/16bit opration (add/sub/div/mul...) *not tested*

There is no document, but C sample code tells how to use.
Have fun!
