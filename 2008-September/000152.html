>  BTW, how can I chop off 6 chars from the BEGINNING of a string, get rid of
>  all the double quotes, and re-print that new string?

By learning assembly language :-) Seriously, if you're going to write
an interpreter, you need be able to do stuff like this with your eyes
closed. Some pseudocode:

    source string in si
    dest string in di
    add si, 6 ; Skip first six chars of source
loop:
    get byte from si
    is byte a double quote?
    inc si and jump back to loop if so
    is byte null?
    quit out if so
    mov byte from si into di
    inc si and di
    back to loop

Again, this is really elementary stuff in assembly. I recommend
reading a good assembly tutorial and getting a good grasp of the
language, otherwise you'll get stuck along the way.
