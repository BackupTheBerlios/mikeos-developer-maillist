Hello, here is current libmikeos status.

http://www.bi.a.u-tokyo.ac.jp/~uaa/gomitext/2008/20080921/20080921.tar.gz

There is four new tests, all API-wrapper of MikeOS
(except os_long_int_negate) have been covered.

	test06	move cursor
	test07	interrupt handler (with INT instruction)
	test08	serial port in/out
	test09	file operation

	----
	test00	"Hello, World!"
	test01	play sound
	test02	date/time string
	test03	string operations
	test04	string input, show/hide cursor
	test05	misc(1)

I will write test code for os_long_int_negate and fundamentals of bcc
(add/subtract/multiply... to int/long int).

And I will release libmikeos-#1 when pile of current sources have passed
all tests.
----

Mike, I am very glad to contribute your OS and merge libmikeos into
source tree. But please check my code carefully before merging.

The libraries included in bcc is under LGPL license, and some parts of
libmikeos' code resemble bcc's library - for example, 16bit division
operation (__idiv.asm).

Does anyone have any good idea to show libmikeos is completely GPL-free?
I think libmikeos should be under MikeOS' license.
