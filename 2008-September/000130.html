Hi all,

I've (finally!) had a proper chance to try this. Excellent work,
Takayoshi! It's great to see that so much of the MikeOS API has been
implemented.

The versions of Bin86 and bcc in the Ubuntu 8.04 repositories don't
work correctly (not a surprise), but after compiling Bin86 from source
everything runs as expected.

Takayoshi, do you have more plans for this, or are you OK to include
it in the next release of MikeOS? I'd like to add a small tutorial to
put it all together, based on the sample programs you've provided.

So, for 1.5 (or 2.0) we will have a great set of new features:

1) Mouse support from Tomasz
2) A C API from Takayoshi
3) FAT12 writing from me

For FAT12, I'm edging closer to the final version. Currently
os_write_file can save files that are readable by the Linux VFAT
driver, but there's a problem inside MikeOS. For single cluster files,
it appears that the Linux driver doesn't consult the FAT table, and
instead takes the sole cluster entry from the directory entry and uses
that. So the problem lies with what I'm doing in the FAT table
sectors. I'm battling on though!
