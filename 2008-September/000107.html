Lo all,

I've been making more headway with FAT12 write support. The attached
program lets you rename files, and the various bits for reading the
root dir entries, writing out the FAT are separated into calls.
Eventually these will become full-on system calls. (Oh, and this also
introduces os_input_dialog, which will be rolled into syscalls.asm for
the next release.)

This is stage 1 done -- next is deletion (finding a root dir entry,
then following and blanking out the chain of cluster entries in the
FAT) and creation (finding a free FAT entry, associating it with a
chain of clusters). So there's still lots to do, but the code is
becoming more modular and straightforward, and hopefully useful to
anyone who's exploring FAT12.
