Been beaving away on this so I'll upload it before I head pubwards!
This code now implements three system calls:

os_remove_file
os_rename_file
os_get_file_size

They depend on various other helper functions, but the code here is
pretty much OS-independent and should be pretty simple to port to
other OSes. (There's a bit of MikeOS-specific code at the start, but
that's just dialog boxes to demonstrate the file delete/remove
functionality.)
